ModularStore is a modern, modular, and fully extensible retail management platform designed for any type of store.
Built with FastAPI, it provides a clean architecture, secure authentication, and a complete set of commercial features that match professional paid systems used in the market today.

This project aims to serve as a generic yet powerful foundation for building retail applications, point-of-sale systems, or full ERP-like solutions.
It can be easily adapted to supermarkets, clothing stores, electronics stores, cosmetic shops, warehouses, and many other business models.
